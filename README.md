# 드림놀이

- 친구 드림복지해주려고 만든 2인용 드림페어세션 ai 시뮬레이터인데 공유해도 좋을것같다싶어서 올려봅니다.
- 제미나이한테 많이 의존해서 이상한 부분이 많을 수도 있습니다. 기본으로 ai는 GPT 5.2로 설정되어 있습니다.
- 모델 바꾸고 싶으시다면 app.py의 gpt-5.2부분을 수정해 주세요
- 크랙을 많이 참고했습니다. 크랙 2인 파티챗과 거의 비슷하게 굴러갑니다(그랬으면 좋겠네요)

## 필요한 것
 - OpenAI API Key: OpenAI API에서 발급 가능합니다(유료!). https://platform.openai.com/docs/quickstart
 - ngrok Auth Token: 외부에서 웹 페이지에 접속하기 위해 필요합니다.(무료!) https://dashboard.ngrok.com/get-started/your-authtoken
   
## 주요 기능
- **스마트 테마**: 세션의 프롤로그와 프롬프트를 분석해서 UI 색이 바뀝니다
- **키워드북**: 세계관 설정이나 캐릭터 정보를 입력해 AI가 기억하도록 합니다. 크랙과 동일한 구조(아마요)
- **프롤로그 & 요약**: 시작 설정과 줄거리를 자유롭게 관리하세요.

## 🚀 시작하기
### 컴퓨터에 못 설치하시겠다 싶으신 분은 맨 아래 방법을 참조해 주세요!
1. 파이썬 설치 (꼭 하셔야 합니다!)
 - 파이썬 공식 다운로드 페이지에 접속합니다. (초보자에게는 3.10이나 3.11 버전을 가장 추천합니다.)
 - 설치 파일을 실행하면 아래 화면이 뜹니다.
 - 가장 중요: 하단의 [Add Python to PATH] 체크박스를 반드시 체크하고 설치하세요! (이걸 안 하면 나중에 명령어가 안 먹힙니다.)

2. 화면 우측 상단의 초록색 [Code] 버튼을 누르고 [Download ZIP]을 클릭하세요.
   
3. 다운로드 받은 압축 파일을 원하는 폴더에 풀어줍니다.
   
4. 필요한 부품 설치 (딱 한 번만 하면 됩니다)
 - 폴더 창 상단의 주소창을 마우스로 클릭합니다.
 - 주소를 싹 지우고 영어로 **cmd**라고 입력한 뒤 엔터(Enter)를 누릅니다.
 - 명령 프롬프트 혹은 터미널이 뜨면 아래 명령어를 복사해서 붙여넣고 엔터를 누르세요:
   ```bash
   pip install -r requirements.txt
   ```
   
5. 키 등록하기 (중요!)
 - 폴더 안에 새 텍스트 문서를 하나 만듭니다.
 - 이름을 `.env`로 바꿔주세요. (뒤에 `.txt`가 붙지 않게 주의! 절대 붙으면 안 됩니다)
 - 만든 파일을 메모장으로 열고 아래 내용을 복사해서 본인의 키 값으로 채운 뒤 저장합니다.
   ```코드 스니펫
   OPENAI_API_KEY=발급받으신openai_API키
   NGROK_AUTH_TOKEN=발급받으신Ngrok토큰
   ADMIN_PASSWORD=3896
   ```
   
6. 프로그램을 실행합니다.
 - 다시 명령 프롬프트 혹은 터미널 창에 아래 명령어를 입력합니다:
   ```bash
   python app.py
   ```
   
- 잠시 기다리면 화면에 나오는 https://xxxx.ngrok-free.app 형태의 주소를 클릭해서 접속하시면 됩니다.
